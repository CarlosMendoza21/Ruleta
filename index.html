GIRAR

Verifican que cada unidad individual de código (como funciones o métodos) funcione correctamente.

Aseguran que diferentes módulos o servicios del software funcionen bien juntos.

Evalúan si el software cumple con los requisitos funcionales especificados.

Verifican que todo el sistema funcione correctamente desde el principio hasta el final, incluyendo todas las interacciones entre componentes y sistemas externos.

Determinan si el sistema completo cumple con los criterios establecidos por el cliente o usuario final.

Evalúan la velocidad, estabilidad y capacidad de respuesta del software bajo diferentes cargas de trabajo para identificar cuellos de botella.

Son una serie de pruebas básicas que se realizan para verificar que las funciones más importantes del software funcionen correctamente.
